# QUESTION TAG SET (FOR BIOMEDICAL KG)

## 1. Core Question Types

- retrieval_basic  
  Direct 1-hop lookup

- clinical_practical  
  Real-world medical usage (treatment, symptoms, contraindications)

- biological_mechanism  
  Pathway / process / molecular-level questions

- reasoning_multihop  
  2–3 hop reasoning across entities

- constraint_filter  
  Questions with filtering conditions (approval, properties, etc.)

- hypothesis_discovery  
  Exploratory / potential / indirect relationships

---

## 2. Aggregation Types

- aggregation_count  
  Counting (how many)

- aggregation_rank  
  Ranking (most, least)

- aggregation_compare  
  Comparison between entities

- aggregation_stat  
  Average, distribution, statistics

- aggregation_extreme  
  Max / min queries

- aggregation_conditional  
  Aggregation with constraints
